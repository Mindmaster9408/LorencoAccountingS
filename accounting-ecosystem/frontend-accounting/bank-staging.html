Confirm Transaction

Move this transaction to the live bank transaction queue. It will appear as Unmatched and ready to allocate.

Reject Transaction

Mark this staged transaction as Rejected. It will NOT be imported into your bank transactions. This action cannot be undone.

Confirm Bank Transfer

This will post a Dr/Cr transfer journal to the General Ledger and move both transactions into the reconciliation queue as matched.

This is a GL-posting operation. It cannot be undone without posting a counter-reversal.

Reject Transfer Suggestion

Dismiss this transfer suggestion. Both transactions will return to the To Review queue for individual actioning. The suggestion is recorded for audit trail purposes.

Reverse Confirmed Transfer

This will reverse the transfer journal in the GL, delete both bank transactions, and return both staging rows to the review queue.

Warning: Cannot reverse if either transaction is already reconciled or has attachments.
